# mica-actuator
用于扩展更多端点，便于在 admin 中进行监控。

## 功能
1. druid star 端点。

## 添加依赖
### maven
```xml
<dependency>
  <groupId>net.dreamlu</groupId>
  <artifactId>mica-actuator</artifactId>
  <version>${version}</version>
</dependency>
```

### gradle
```groovy
compile("net.dreamlu:mica-actuator:${version}")
```
